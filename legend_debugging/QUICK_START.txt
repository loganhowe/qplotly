QPLOTLY LEGEND FEATURES - QUICK START GUIDE
===========================================

This folder contains 15 demonstration plots showing qplotly's matplotlib-compatible
legend functionality.

BEST PLOTS TO START WITH:
-------------------------

1. 01_basic_default.png
   - Shows the default legend behavior
   - Upper right position, opaque white box, black border
   - This is what you get by just calling fig.legend()

2. 02_all_locations_grid.png
   - Shows all 11 matplotlib location names in one view
   - Proves all positions work and stay inside plot boundaries

3. 10_style_comparison.png
   - 2x2 grid comparing different legend styles
   - Quick visual reference for styling options

4. 07_many_traces_auto_colors.png
   - Shows 8 traces with auto-color scheme
   - Demonstrates legend with nipy_spectral colors

KEY FEATURES:
-------------
✓ Default: Upper right, opaque white, black border
✓ All 11 matplotlib location names supported
✓ Legends stay inside plot boundaries (x: 0.02-0.98, y: 0.02-0.98)
✓ Custom colors (facecolor, edgecolor)
✓ Transparency control (framealpha)
✓ Border on/off (frameon)
✓ Font size control
✓ Works with auto-color scheme
✓ Works with all plot types


MATPLOTLIB COMPATIBILITY:
--------------------------
These parameters work exactly like matplotlib:
  - loc (location name string)
  - fontsize
  - frameon (border on/off)
  - framealpha (transparency)
  - facecolor (background color)
  - edgecolor (border color)


QUICK CODE EXAMPLES:
--------------------

# Default legend
fig.legend()

# Different position
fig.legend(loc='lower left')

# Custom colors
fig.legend(loc='upper right', facecolor='lightblue', edgecolor='blue')

# Transparent
fig.legend(loc='center', framealpha=0.5)

# No border
fig.legend(loc='best', frameon=False)

# Large font
fig.legend(loc='upper right', fontsize=16)

python demo_all_features.py
